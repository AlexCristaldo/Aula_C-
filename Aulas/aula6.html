Lista

Nesta aula será estudada a estrutura Lista, utilizada para armazenar dados de forma dinâmica e organizada.

Vídeo da Aula



  • Compreender listas.
  • Inserir elementos.
  • Remover elementos.
  • Percorrer listas.

Listas são estruturas dinâmicas que permitem inserções e remoções de elementos com maior flexibilidade do que vetores.

Curiosidade:

Redes sociais utilizam estruturas semelhantes a listas para organizar feeds e sequências de publicações.

Exercício 1

Crie uma lista e adicione os valores 10, 20 e 30. Mostre todos os elementos.

using namespace std;

}
