Variáveis e Operadores Lógicos

Nesta aula serão apresentados os conceitos fundamentais da linguagem C++. O aluno aprenderá como armazenar informações em variáveis e utilizar operadores lógicos para realizar comparações e tomadas de decisão.

Vídeo da Aula

    Objetivo:

  • Compreender o conceito de variável.
  • Conhecer os principais tipos de dados.
  • Utilizar operadores aritméticos.
  • Aprender operadores lógicos.

Variáveis são espaços reservados na memória para armazenar informações. Entre os principais tipos de dados estão int, float, char e bool. Também foram apresentados os operadores lógicos AND (&&), OR (||) e NOT (!), utilizados em expressões condicionais.



Exercícios

Exercício 1

Crie um programa que leia dois números e mostre a soma.

Ver Resolução 
#include <iostream>
using namespace std;

int main() {

    int a, b;

    cin >> a;
    cin >> b;

    cout << a + b;

    return 0;
}

Exercício 2

Crie um programa que leia a idade de uma pessoa e mostre a idade informada.

Ver Resolução 
    
#include <iostream>
using namespace std;

int main() {

    int idade;

    cout << "Digite sua idade: ";
    cin >> idade;

    cout << "Sua idade e: " << idade;

    return 0;
}

Exercício 3

Crie um programa que leia dois números e mostre a soma, subtração, multiplicação e divisão.

Ver Resolução 

#include <iostream>
using namespace std;

int main() {

    float a, b;

    cin >> a;
    cin >> b;

    cout << "Soma: " << a + b << endl;
    cout << "Subtracao: " << a - b << endl;
    cout << "Multiplicacao: " << a * b << endl;
    cout << "Divisao: " << a / b << endl;

    return 0;
}
