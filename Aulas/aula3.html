Vetores

Nesta aula será apresentado o conceito de vetores, estruturas que permitem armazenar vários valores do mesmo tipo em uma única variável.

Vídeo da Aula



    Objetivo:

  • Declarar vetores.
  • Armazenar dados em vetores.
  • Percorrer vetores com laços.
  • Manipular elementos armazenados.

Vetores são estruturas de dados que armazenam múltiplos valores utilizando índices. Eles facilitam o armazenamento e manipulação de grandes quantidades de informações.

Curiosidade:

Os índices dos vetores em C++ começam em zero, característica herdada da linguagem C.

Exercícios

Exercício 1

Crie um vetor com 5 posições, leia os 5 valores e mostre todos eles.

Ver Resolução 
    
#include 
using namespace std;

int main() {

    int vet[5];

    for(int i = 0; i < 5; i++){
        cin >> vet[i];
    }

    for(int i = 0; i < 5; i++){
        cout << vet[i] << endl;
    }

    return 0;
}

Exercício 2

Leia 10 números em um vetor e mostre a soma de todos os elementos.

Ver Resolução 

#include 
using namespace std;

int main() {

    int vet[10];
    int soma = 0;

    for(int i = 0; i < 10; i++){
        cin >> vet[i];
        soma += vet[i];
    }

    cout << "Soma: " << soma;

    return 0;
}

Exercício 3

Leia 5 números e mostre o maior valor armazenado no vetor.

Ver Resolução 

#include 
using namespace std;

int main() {

    int vet[5];

    for(int i = 0; i < 5; i++){
        cin >> vet[i];
    }

    int maior = vet[0];

    for(int i = 1; i < 5; i++){
        if(vet[i] > maior){
            maior = vet[i];
        }
    }

    cout << "Maior valor: " << maior;

    return 0;
}

Exercício 4

Leia 8 números e conte quantos deles são pares.

Ver Resolução 

#include 
using namespace std;

int main() {

    int vet[8];
    int pares = 0;

    for(int i = 0; i < 8; i++){
        cin >> vet[i];

        if(vet[i] % 2 == 0){
            pares++;
        }
    }

    cout << "Quantidade de pares: " << pares;

    return 0;
}

Exercício 5

Leia 6 números e mostre os elementos na ordem inversa.

Ver Resolução 

#include 
using namespace std;

int main() {

    int vet[6];

    for(int i = 0; i < 6; i++){
        cin >> vet[i];
    }

    for(int i = 5; i >= 0; i--){
        cout << vet[i] << endl;
    }

    return 0;
}

Exercício 6

Leia um vetor com 5 números e um número qualquer. Verifique se esse número existe no vetor.

Ver Resolução 

#include 
using namespace std;

int main() {

    int vet[5];
    int procura;
    bool encontrou = false;

    for(int i = 0; i < 5; i++){
        cin >> vet[i];
    }

    cin >> procura;

    for(int i = 0; i < 5; i++){
        if(vet[i] == procura){
            encontrou = true;
        }
    }

    if(encontrou){
        cout << "Numero encontrado";
    }
    else{
        cout << "Numero nao encontrado";
    }

    return 0;
}
