Struct

Nesta aula será estudada a estrutura Struct, utilizada para agrupar diferentes informações em um único tipo de dado.

Vídeo da Aula



  • Criar structs.
  • Armazenar múltiplos atributos.
  • Organizar dados.
  • Utilizar vetores de structs.

Structs permitem agrupar diferentes tipos de dados relacionados, facilitando a modelagem de entidades como alunos, produtos, funcionários e veículos.

Curiosidade:

As structs são amplamente utilizadas em sistemas reais para representar objetos do mundo real antes mesmo do uso de Programação Orientada a Objetos.

Exercícios

Exercício 1

Crie uma struct chamada Pessoa contendo nome e idade. Mostre os dados na tela.

Ver Resolução 
    
#include 
using namespace std;

struct Pessoa {
    string nome;
    int idade;
};

int main() {

    Pessoa p;

    p.nome = "Joao";
    p.idade = 25;

    cout << p.nome << endl;
    cout << p.idade;

    return 0;
}

Exercício 4

Crie um vetor de 3 alunos utilizando struct e mostre os nomes cadastrados.

Ver Resolução 

#include 
using namespace std;

struct Aluno {
    string nome;
};

int main() {

    Aluno alunos[3];

    for(int i = 0; i < 3; i++){
        cin >> alunos[i].nome;
    }

    for(int i = 0; i < 3; i++){
        cout << alunos[i].nome << endl;
    }

}

Exercício 5

Crie uma struct Carro contendo marca e ano. Cadastre dois carros e mostre suas informações.

Ver Resolução 

#include 
using namespace std;

struct Carro {
    string marca;
    int ano;
};

int main() {

    Carro carros[2];

    for(int i = 0; i < 2; i++){
        cin >> carros[i].marca;
        cin >> carros[i].ano;
    }

    for(int i = 0; i < 2; i++){
        cout << carros[i].marca << " ";
        cout << carros[i].ano << endl;
    }

    return 0;
}

Exercício 6

Crie uma struct Funcionario contendo nome e salário. Mostre qual funcionário possui o maior salário.

Ver Resolução 

#include 
using namespace std;

struct Funcionario {
    string nome;
    float salario;
};

int main() {

    Funcionario f[2];

    for(int i = 0; i < 2; i++){
        cin >> f[i].nome;
        cin >> f[i].salario;
    }

    if(f[0].salario > f[1].salario){
        cout << f[0].nome;
    }
    else{
        cout << f[1].nome;
    }

    return 0;
}
